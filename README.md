#### Data Visualization

### Objective
To create a bar chart to visualize the distribution of a continuous variable.

### Tool Used
- MS Excel

### Dataset
- World Bank Population Dataset

### Visualization
![Population Chart](population_chart_2020.png)

- Processed Excel file: population_data_analysis.xlsx


### Explanation
The bar chart shows the population distribution of selected countries for the year 2020. It enables a clear comparison of population sizes across different countries and highlights the significant differences in population levels.
